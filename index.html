BOI Blockchain Object Identifier System

A decentralized digital object identifier system based on the NEM blockchain.

What is BOI

Clicking on a BOI link (BOI:Proce./1540543147317) takes you to one related to a single resource. If the URLs or services change over time, e.g., the resource moves, this same DOI will continue to resolve to the correct resources or services at their new locations. BOI also links metadata to the object.

Build on NEM

The BOI-system is based on the NEM blockchain and open source, which means its 100% decentralized and immutable. No centralized entity controls the system. Everyone can start to create BOIâ€™s for their YouTube content, academic publications or digital art.

Price

The price of using the BOI-system is only a fraction of using traditional systems. You can register a BOI as low as 0.1 xem. The cost that is paid is to send a transaction on the blockchain network.

How it works

The BOI data and ownership is stored on the blockchain.

As a publisher

When you create a BOI it is translated to an address on the blockchain. By sending a transaction including the metadata as a json object to the BOI address ownership is established and the data is saved in the blockchain.

As a user

When clicking a BOI link a request to the blockchain will retrieve the related metadata. You can use a service such as ours or make your own request using the NEM api.

Example

Jack is a publisher and want to create a BOI for his published work.

Creating

Jack fill in the published object info, generates an BOI NEM address. He sends the generated message (json object) from his account to the generated BOI address. Now the BOI is active.

Updating

The original URL provided in the BOI is outdated. So Jack sends an message (json object) with the updated info to the BOI NEM address.

API

Your can use the API to retrive info from Jacks BOI. Either by direct linking or for the metadata.
